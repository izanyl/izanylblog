Screenprints

Model with identity Brand *Graffiti* (CMYK)

This scrteenprint looks like a real life portrait due to the CMYK process. We print in the order of Cyan, Magenta, Yellow, & Black We do this by first seperating the colors of the original image in Photoshop and then exposing & printing those separate colors in 4 different screens. It is a much longer process but those extra steps can be a handful. Eventually, the colors mix in on top of each other which result in creating a realistic effect. It is always best to work with an image that contains vivid colors in order to create a constrast and vibrant screenprint.

This screenprint was made with the intention to have my identity username on the background within a graffiti wall. The original image was found online and did not contain the same colors used in this print. The original image only contains a brown graffiti wall, and the model wearing a white shirt. I chose to select an online image because I was never able to get a hold of someone to model for this project idea. On top of That I did not intend to graffiti a random wall. So I Photoshoped the name on the wall and changed as much as I could, with the usage of colors in order to have vivid colors.

(H14 x W16)in.

Self Portrait with Graffiti (CMYK)

This CMYK print was done during the same time as the 'Model with Identity Brand' and was inspired by the same idea as that print. This is an old picture of myself that was taken of me in 2018. I have taken many photographs that contain a graffiti background, but this is the only daytime one. Since it has many vibrant colors, I decided to do this with the CMYK Process. I also decided to show the borders which show two circles that are supposed to be registration marks for when we print the CMYK colors on top of each other. The goal is to line up the prints in order to get the best quality image. However, if you misalign the image, it can result in having an error in the print.

(H10 x W12)in.
